Input:<br />
&ensp;&ensp;&ensp;&ensp;p, principal amount<br />
&ensp;&ensp;&ensp;&ensp;t, time period in years<br />
&ensp;&ensp;&ensp;&ensp;r, annual rate of interest<br />
Output<br />
&ensp;&ensp;&ensp;&ensp;simple interest = p*t*r
